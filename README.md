# Curso de Programación en Python

Este curso proporciona una introducción comprensiva a la programación en Python, desde conceptos básicos hasta temas avanzados.

## Estructura del Curso

### Semana 0: Funciones, Variables
- Conceptos fundamentales
- Introducción a Python y su sintaxis
- Funciones básicas: `print()`, `input()`
- Variables y asignación
- Tipos de datos básicos: strings, integers, floats
- Comentarios en el código
- Formateo de strings (f-strings)
- Métodos de strings: `strip()`, `title()`, `capitalize()`, `split()`
- Operadores aritméticos básicos
- Conversión de tipos (type casting)
- **Proyecto**: Calculadora de propinas o programa interactivo simple

### Semana 1: Condicionales
- Control de flujo
- Operadores de comparación: `>`, `<`, `>=`, `<=`, `==`, `!=`
- Sentencias `if`, `elif`, `else`
- Operadores lógicos: `and`, `or`, `not`
- Operador `match` (switch en Python 3.10+)
- Valores booleanos y expresiones booleanas
- Anidamiento de condicionales
- **Proyecto**: Sistema de calificaciones o validador de entrada

### Semana 2: Bucles (Loops)
- Estructuras de repetición
- Bucle `while`
- Bucle `for`
- Iteración sobre secuencias
- Función `range()`
- Sentencias `break` y `continue`
- Bucles infinitos y cómo evitarlos
- Validación de entrada con bucles
- **Proyecto**: Juego de adivinanzas o generador de patrones

### Semana 3: Excepciones
- Manejo de errores
- Tipos de errores comunes: `SyntaxError`, `ValueError`, `NameError`
- Bloques `try` y `except`
- Cláusula `else` en manejo de excepciones
- Cláusula `finally`
- Creación de excepciones con `raise`
- Buenas prácticas en manejo de errores
- **Proyecto**: Validador robusto de datos de usuario

### Semana 4: Librerías
- Módulos y paquetes
- Importación de módulos: `import`
- Importación selectiva: `from ... import ...`
- Librería estándar de Python
- Módulos útiles: `random`, `statistics`, `sys`, `os`
- Uso de `pip` para instalar paquetes externos
- Creación de módulos propios
- Documentación y búsqueda de librerías
- **Proyecto**: Programa que utiliza múltiples librerías externas

### Semana 5: Pruebas Unitarias
- Testing y depuración
- Importancia del testing
- Framework `pytest`
- Función `assert`
- Estructura de pruebas unitarias
- Casos de prueba: normales, límite, excepciones
- Test-Driven Development (TDD) básico
- Cobertura de código
- **Proyecto**: Crear funciones con sus respectivas pruebas unitarias

### Semana 6: Entrada/Salida de Archivos
- Manipulación de archivos
- Función `open()`
- Modos de apertura: `r`, `w`, `a`, `r+`
- Lectura de archivos: `read()`, `readline()`, `readlines()`
- Escritura en archivos: `write()`, `writelines()`
- Bloque `with` para manejo automático de archivos
- Archivos CSV: lectura y escritura
- Módulo `csv`
- Trabajo con rutas de archivos
- **Proyecto**: Agenda de contactos o analizador de datos CSV

### Semana 7: Expresiones Regulares
- Patrones y búsqueda de texto
- Introducción a regex
- Módulo `re`
- Funciones: `search()`, `match()`, `findall()`, `sub()`
- Patrones básicos y metacaracteres
- Clases de caracteres: `\d`, `\w`, `\s`
- Cuantificadores: `*`, `+`, `?`, `{n,m}`
- Grupos de captura
- Validación de formatos (emails, teléfonos, etc.)
- **Proyecto**: Validador de formatos o extractor de información

### Semana 8: Programación Orientada a Objetos
- POO en Python
- Clases y objetos
- Método `__init__` (constructor)
- Atributos de instancia y de clase
- Métodos de instancia
- Encapsulación y convenciones de privacidad
- Propiedades con `@property`
- Métodos especiales: `__str__`, `__repr__`
- Herencia
- Métodos de clase `@classmethod`
- Métodos estáticos `@staticmethod`
- **Proyecto**: Sistema de gestión (biblioteca, inventario, estudiantes)

### Semana 9: Et Cetera
- Conceptos avanzados
- Conjuntos (sets)
- Diccionarios avanzados
- Funciones lambda
- List comprehensions
- Dictionary comprehensions
- Argumentos variables: `*args` y `**kwargs`
- Generadores e iteradores
- Decoradores básicos
- Manejo de APIs simples
- Introducción a entornos virtuales
- **Proyecto Final**: Aplicación completa que integre múltiples conceptos

## Material del Curso

El material del curso se encuentra organizado en carpetas semanales (S0-S9), cada una conteniendo el material correspondiente a esa semana.

## Requisitos

- Python 3.10 o superior
- Editor de texto o IDE (recomendado: Visual Studio Code)
- Conexión a internet para la instalación de paquetes

## Metodología

Cada semana combina:
- Teoría y conceptos fundamentales
- Ejemplos prácticos
- Ejercicios guiados
- Proyecto semanal
- Material complementario

## Evaluación

- Problem sets: 60%
- Proyecto final: 30%
- Participación: 10%
